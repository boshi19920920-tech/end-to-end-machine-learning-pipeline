# Data

This project uses a publicly available diabetes prediction dataset for binary classification.

The raw dataset is not included in this repository unless the dataset license clearly allows redistribution. To reproduce the analysis, download the dataset from Kaggle and place the CSV file in this folder.

Suggested file path: data/diabetes_prediction_dataset.csv

The dataset includes demographic, clinical, and lifestyle related predictors. The outcome variable indicates whether an individual has diabetes.

Main variables may include gender, age, hypertension, heart disease, smoking history, BMI, HbA1c level, blood glucose level, and diabetes status.

The analysis notebook will load the dataset from this folder and perform data cleaning, exploratory analysis, feature engineering, model training, evaluation, calibration, and interpretation.
